Analysis of Director Genders in Popular Films
Feb 25, 2024 7 minutes

Check out the Github for this project Check out the Shiny app Overview After a bit of recent uproar around the director for the Barbie film not recieving an oscar award, I decided that it might be interesting to do a project analysing the disparity between genders of directors in popular films. When deciding which dataset to choose, I checked out a few different film sites and their rating systems. I settled on Letterboxd because it seemed to be the site with the highest number of user watches and reviews for most films compared to sites like IMDB and TMDB.

Sentiment Analysis of Financial News Headlines with Python - Part 3
Feb 18, 2024 8 minutes

Check out the Github for this project This post is the first of a 3 part series. Part 1 - Scraping financial news headlines. Part 2 - Performing sentiment analysis to get sentiment scores for each headline. Part 3 - Aggregating our headline sentiment scores to create a sentiment value for each day the market is open, and checking for correlation between this and the stock price movement over the following day.
